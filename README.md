# CHZ Bank

![](/doc/images/ui-branch.png)


## 使用方法
### 获取代码
```
git colne https://github.com/charleyustccs/CHZ_Bank.git
```

### 安装项目依赖
```
yarn
```
或者
```
yarn install
```

### 运行
```
yarn start
```
一般情况下自动在浏览器中打开 http://localhost:3000/

`注：使用npm或cnpm也行`
